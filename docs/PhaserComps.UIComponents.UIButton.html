Class: UIButton

PhaserComps.UIComponents. UIButton

Button component prototype, has states up, over, down, disable Emits EVENT_CLICK on click. When disabled, doesn't interact to mouse events and move to state disable

new UIButton( [parent] [, key] [, labelText])
